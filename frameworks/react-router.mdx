## Get started with a template

The easiest way to get started is to use the **React Router v7 + TypeScript** template. It comes preconfigured with Panda CSS, Ark UI, Fidely UI v2, and theme support.

```bash
npx gitpick https://github.com/fidely-ui/fidely-templates/tree/main/react-router-template
```
