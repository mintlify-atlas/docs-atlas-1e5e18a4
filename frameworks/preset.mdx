## Get started quickly

The easiest way to get started is to use the **Vite starter**.

<Note>
  The command below uses Fidely UI v1. It will be updated once v2 is stable.
</Note>

```bash
npx gitpick https://github.com/fidely-ui/fidely-templates/tree/main/panda-preset-starter
```

---

## Manual installation

If you prefer setting things up yourself, follow the steps below.

<Steps>
  <Step title="Install Panda CSS">
    ```bash
    npm install -D @pandacss/dev
    npx panda init --postcss
    ```
  </Step>

  <Step title="Install the preset">
    ```bash
    npm install @fidely-ui/panda-preset
    ```

    <Info>
      This preset contains design tokens, recipes, and utilities. No React components are included. You can use your own components or another UI library while still leveraging the preset.
    </Info>
  </Step>

  <Step title="Update package.json scripts">
    Open your **package.json** file and add the `prepare` script:

    ```json
    {
      "scripts": {
        "prepare": "panda codegen",
        "dev": "vite",
        "build": "vite build",
        "preview": "vite preview"
      }
    }
    ```
  </Step>

  <Step title="Update tsconfig.app.json">
    Update your `compilerOptions` to include the following:

    ```json
    {
      "compilerOptions": {
        "target": "ESNext",
        "module": "ESNext",
        "moduleResolution": "Bundler",
        "skipLibCheck": true,
        "paths": {
          "@/*": ["./src/*"],
          "@styled-system/*": ["./styled-system/*"]
        }
      }
    }
    ```
  </Step>

  <Step title="Configure Vite path aliases">
    Vite does not automatically read **tsconfig.json** path aliases. You must manually configure them in **vite.config.ts** so Panda's `styled-system` imports resolve correctly.

    ```ts
    import { defineConfig } from 'vite'
    import react from '@vitejs/plugin-react'
    import path from 'node:path'

    export default defineConfig({
      plugins: [react()],
      resolve: {
        alias: {
          '@': path.resolve(__dirname, './src'),
          '@styled-system': path.resolve(__dirname, './styled-system'),
        },
      },
    })
    ```
  </Step>

  <Step title="Configure Panda CSS">
    Update **panda.config.ts**:

    ```ts
    import { defineConfig } from '@pandacss/dev'
    import { fidelyPreset } from '@fidely-ui/panda-preset'

    export default defineConfig({
      presets: [fidelyPreset()],

      // Enable CSS reset
      preflight: true,

      // Files Panda should scan for styles
      include: ['./src/**/*.{js,jsx,ts,tsx}'],

      exclude: [],

      // Theme customization
      theme: {
        extend: {},
      },

      // Generate static CSS for all recipes
      staticCss: {
        recipes: '*',
      },

      // Output directory for generated styles
      outdir: 'styled-system',

      jsxFramework: 'react',

      // Remove Panda default 50 / 950 color steps
      plugins: [
        {
          name: 'panda-headless-colors',
          hooks: {
            'preset:resolved': ({ utils, preset, name }) => {
              if (name === '@pandacss/preset-panda') {
                return utils.omit(preset, [
                  'theme.tokens.colors',
                  'theme.semanticTokens.colors',
                ])
              }
              return preset
            },
          },
        },
      ],
    })
    ```

    After updating the config, generate the styled system:

    ```bash
    npx panda codegen
    ```

    This creates the `styled-system` folder with:

    - Design tokens (colors, spacing, fonts, etc.)
    - CSS recipes (buttons, inputs, cards, etc.) and more
  </Step>

  <Step title="Configure the entry CSS with layers">
    Open **src/App.css** and replace its contents with:

    ```css
    @layer reset, base, tokens, recipes, utilities;
    ```

    <Tip>You can safely remove unused CSS files and delete **index.css** if it is present.</Tip>
  </Step>

  <Step title="Import global CSS in the app entry">
    Update **src/main.tsx**:

    ```tsx
    import { StrictMode } from 'react'
    import { createRoot } from 'react-dom/client'

    import './App.css'
    import App from './App'

    createRoot(document.getElementById('root')!).render(
      <StrictMode>
        <App />
      </StrictMode>
    )
    ```
  </Step>
</Steps>

## Use your own components

You can consume the preset styles in your own components using the `styled` factory:

```tsx
import { styled } from '@styled-system/jsx'

const MyButton = styled('button', {
  base: {
    bg: 'orange.9',
    color: 'white',
    px: '4',
    py: '2',
    borderRadius: 'md',
    _hover: { bg: 'orange.11' },
  },
})

export function Demo() {
  return <MyButton>Click me</MyButton>
}
```

Or use recipes directly:

```tsx
import { button } from '@styled-system/recipes'

export function Demo() {
  return <button className={button({ size: 'lg' })}>Click me</button>
}
```

## Notes

- All recipes provided by the preset are available in `styled-system` after codegen.
- You can extend the theme under `theme.extend` in `panda.config.ts`.
- No Fidely UI React components are included — the preset is purely tokens, recipes, and utilities.
