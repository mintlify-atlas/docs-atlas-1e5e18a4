## Get started with a template

The easiest way to get started is to use the **Laravel v12 + Inertia.js + React** template. It comes preconfigured with Panda CSS, Ark UI, Fidely UI v2, and theme support.

```bash
npx gitpick https://github.com/fidely-ui/fidely-templates/tree/main/laravel-template
```
