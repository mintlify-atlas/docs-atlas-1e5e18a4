Fidely UI is a React component library built on [Ark UI](https://ark-ui.com/) headless primitives and [Panda CSS](https://panda-css.com/) zero-runtime styling. It gives you 50+ accessible, fully themeable components — ready to use in production without sacrificing flexibility or performance.

## Get started in minutes

<Steps>
  <Step title="Install the packages">
    Install `@fidely-ui/react` and `@fidely-ui/panda-preset` using your preferred package manager.

    <CodeGroup>
      ```bash npm
      npm install @fidely-ui/react @fidely-ui/panda-preset
      ```
      ```bash pnpm
      pnpm add @fidely-ui/react @fidely-ui/panda-preset
      ```
      ```bash yarn
      yarn add @fidely-ui/react @fidely-ui/panda-preset
      ```
    </CodeGroup>
  </Step>
  <Step title="Configure Panda CSS">
    Add the Fidely preset to your `panda.config.ts`:

    ```ts panda.config.ts
    import { defineConfig } from '@pandacss/dev'
    import { fidelyPreset } from '@fidely-ui/panda-preset'

    export default defineConfig({
      presets: [fidelyPreset()],
      preflight: true,
      include: [
        './src/**/*.{js,jsx,ts,tsx}',
        './node_modules/@fidely-ui/react/dist/panda.buildinfo.json',
      ],
      outdir: 'styled-system',
      jsxFramework: 'react',
      staticCss: { recipes: '*' },
    })
    ```
  </Step>
  <Step title="Use your first component">
    Import and render any component from `@fidely-ui/react`:

    ```tsx
    import { Button, Stack, Text } from '@fidely-ui/react'

    export default function App() {
      return (
        <Stack gap="4" p="8">
          <Text fontSize="xl" fontWeight="bold">Hello, Fidely UI!</Text>
          <Button variant="solid">Get Started</Button>
        </Stack>
      )
    }
    ```
  </Step>
</Steps>

## Why Fidely UI?

<CardGroup cols={2}>
  <Card title="Accessible by default" icon="universal-access">
    Every component inherits Ark UI's ARIA-compliant headless foundation — keyboard navigation, screen reader support, and focus management included.
  </Card>
  <Card title="Zero-runtime styling" icon="bolt">
    Panda CSS generates static CSS at build time — no runtime overhead, smaller bundles, and fully typed design tokens.
  </Card>
  <Card title="Fully themeable" icon="swatchbook">
    Choose from 25+ built-in accent colors and 6 gray scales. Override border radii, typography, spacing, and component recipes to match your brand.
  </Card>
  <Card title="Composable and extensible" icon="layer-group">
    Use the `fidely` factory to create styled Ark UI primitives. Extend or override any component recipe without losing flexibility.
  </Card>
</CardGroup>
