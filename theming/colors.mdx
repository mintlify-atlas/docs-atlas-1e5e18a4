Fidely UI's color system is built on top of **[Radix Colors](https://www.radix-ui.com/colors)**, providing perceptually balanced scales that work well in both light and dark modes. Every color is available as a `ColorPalette` object that carries both raw tokens and semantic tokens.

## Gray colors

There are 6 gray scales available. Each can be passed as `grayColor` in `fidelyPreset()`. The gray color drives all `bg.*`, `fg.*`, and `border.*` semantic tokens.

| Name | Character |
|---|---|
| `neutral` | Pure gray, no hue cast |
| `mauve` | Subtle purple tint |
| `olive` | Subtle green tint |
| `sage` | Muted green, softer than olive |
| `sand` | Warm sandy tint |
| `slate` | Blue-gray, popular for modern UIs |

```ts panda.config.ts
import { defineConfig } from '@pandacss/dev'
import { fidelyPreset } from '@fidely-ui/panda-preset'
import slate from '@fidely-ui/panda-preset/colors/slate'
import teal from '@fidely-ui/panda-preset/colors/teal'

export default defineConfig({
  presets: [
    fidelyPreset({
      accentColor: teal,
      grayColor: slate,
    }),
  ],
})
```

## Color pairing

Fidely UI lets you combine any gray and accent to shape your product's visual personality. By default, `neutral` is used for both, giving a clean minimal look. Swap in any color from the palette to differentiate your brand.

## Using color tokens directly

Every color scale has 12 numbered steps plus alpha variants. Use them as style prop values:

```tsx
// Step-based usage
<Box bg="teal.3" color="teal.11" borderColor="teal.6" />

// Alpha variant
<Box bg="teal.alpha3" />
```

The scale steps follow the Radix Colors convention:

| Steps | Use case |
|---|---|
| 1–2 | App backgrounds |
| 3–5 | Component backgrounds |
| 6–8 | Borders and separators |
| 9–10 | Solid fills (buttons, badges) |
| 11–12 | Text and high-contrast fills |

## Importing a color directly

You can import any color and use it as a `ColorPalette` value in your config or pass its tokens to utilities like `createColorTokens`:

```ts
import orange from '@fidely-ui/panda-preset/colors/orange'
import crimson from '@fidely-ui/panda-preset/colors/crimson'
import neutral from '@fidely-ui/panda-preset/colors/neutral'

// orange.name      → 'orange'
// orange.tokens    → raw Panda color tokens
// orange.semanticTokens → semantic token mappings
```

<Note>
  All colors exported from `@fidely-ui/panda-preset/colors/*` implement the `ColorPalette` interface from `types.ts`, which requires `name`, `tokens`, and `semanticTokens`.
</Note>
