## Overview

Fidely UI is built on top of **Panda CSS** and **Ark UI**. It ships as two packages:

<CardGroup cols={2}>
  <Card title="@fidely-ui/react" icon="react">
    The React component library. Includes all accessible, headless-first UI components styled with Panda CSS.
  </Card>
  <Card title="@fidely-ui/panda-preset" icon="palette">
    The Panda CSS preset. Provides design tokens, slot recipes, and utilities. Use this independently or as the foundation for your own components.
  </Card>
</CardGroup>

---

## Install packages

Install both packages using your preferred package manager.

<CodeGroup>

```bash npm
npm install @fidely-ui/react @fidely-ui/panda-preset
```

```bash pnpm
pnpm add @fidely-ui/react @fidely-ui/panda-preset
```

```bash yarn
yarn add @fidely-ui/react @fidely-ui/panda-preset
```

</CodeGroup>

<Note>
  `@fidely-ui/react` and `@fidely-ui/panda-preset` require `react >= 18` and `@pandacss/dev >= 1.1.0` as peer dependencies.
</Note>
